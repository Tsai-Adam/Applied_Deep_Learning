# Applied Deep Learning

This repository contains coursework projects for the **Applied Deep Learning** module.

The purpose of these assignments is to explore how deep learning models can be applied to solve prediction problems and how machine learning techniques can support potential business applications.

## Coursework 1

In this project, we developed a deep learning model using **TensorFlow / Keras** to solve a prediction problem based on a dataset from Kaggle.

The objectives of this coursework include:

- Data preprocessing and exploration  
- Feature engineering  
- Designing and training a neural network model  
- Evaluating model performance using appropriate metrics  
- Discussing potential business applications of the model  

The project demonstrates the complete workflow of a deep learning project, from data preparation to model evaluation.

---

## Coursework 2

The second coursework focuses on improving and extending the deep learning approach.

This project includes:

- experimenting with different neural network architectures  
- tuning hyperparameters  
- improving model performance  
- analysing model results  
- discussing how the model could be applied in real-world scenarios
